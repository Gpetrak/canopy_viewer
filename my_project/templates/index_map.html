This is a project that allows you to explore the Earth, using vector and raster data. You can display and analyze satellite images, that have been processed with Remote Sensing techniques. Also, you can modify vector data using GIS techniques like buffer, intersection, union etc.

Note: For the WPS processing you have to choose the "WPS processing" layer clicking the (+) sign on the right side of the map.
